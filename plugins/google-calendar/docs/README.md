## 快速参考

### 七大 API 模块

1. **Events** - 事件的增删改查、重复事件、与会者、会议等
2. **Calendars** - 日历元数据、主/次要日历、创建与删除
3. **CalendarList** - 用户日历列表、颜色、提醒、通知
4. **Settings** - 用户偏好（时区、语言、日期格式等）
5. **ACL** - 访问控制、共享与权限
6. **Colors** - 日历/事件颜色调色板
7. **FreeBusy** - 忙闲查询
