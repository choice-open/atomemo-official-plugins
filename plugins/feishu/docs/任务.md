## API 列表

### 任务管理

| 序号 | API 名称         | Method   | Path                                                 | 备注 |
| ---- | ---------------- | -------- | ---------------------------------------------------- | ---- |
| 1    | 创建任务         | `POST`   | `/open-apis/task/v2/tasks`                           |      |
| 2    | 更新任务         | `PATCH`  | `/open-apis/task/v2/tasks/:task_guid`                |      |
| 3    | 获取任务详情     | `GET`    | `/open-apis/task/v2/tasks/:task_guid`                |      |
| 4    | 删除任务         | `DELETE` | `/open-apis/task/v2/tasks/:task_guid`                |      |
| 5    | 添加任务成员     | `POST`   | `/open-apis/task/v2/tasks/:task_guid/add_members`    |      |
| 6    | 移除任务成员     | `POST`   | `/open-apis/task/v2/tasks/:task_guid/remove_members` |      |
| 7    | 列取任务列表     | `GET`    | `/open-apis/task/v2/tasks`                           |      |
| 8    | 列取任务所在清单 | `GET`    | `/open-apis/task/v2/tasks/:task_guid/tasklists`      |      |
| 9    | 任务加入清单     | `POST`   | `/open-apis/task/v2/tasks/:task_guid/add_tasklist`   |      |
| 10   | 任务移出清单     | `POST`   | `/open-apis/task/v2/tasks/:task_guid/remove_tasklist` |      |
| 11   | 添加任务提醒     | `POST`   | `/open-apis/task/v2/tasks/:task_guid/add_reminders`  |      |
| 12   | 移除任务提醒     | `POST`   | `/open-apis/task/v2/tasks/:task_guid/remove_reminders` |      |

### 清单管理

| 序号 | API 名称       | Method   | Path                                                       | 备注 |
| ---- | -------------- | -------- | ---------------------------------------------------------- | ---- |
| 13   | 创建清单       | `POST`   | `/open-apis/task/v2/tasklists`                             |      |
| 14   | 获取清单详情   | `GET`    | `/open-apis/task/v2/tasklists/:tasklist_guid`              |      |
| 15   | 更新清单       | `PATCH`  | `/open-apis/task/v2/tasklists/:tasklist_guid`              |      |
| 16   | 删除清单       | `DELETE` | `/open-apis/task/v2/tasklists/:tasklist_guid`              |      |
| 17   | 添加清单成员   | `POST`   | `/open-apis/task/v2/tasklists/:tasklist_guid/add_members`  |      |
| 18   | 移除清单成员     | `POST`   | `/open-apis/task/v2/tasklists/:tasklist_guid/remove_members` |      |
| 19   | 获取清单任务列表 | `GET`    | `/open-apis/task/v2/tasklists/:tasklist_guid/tasks`         |      |
| 20   | 获取清单列表   | `GET`    | `/open-apis/task/v2/tasklists`                             |      |

## 说明

- 上述接口均来自飞书开放平台服务端 API（Task v2）范畴，按「任务」业务场景整理。
- `任务成员`、`任务加/移清单`、`任务提醒`、`清单成员`在 Task v2 中为动作型接口，统一使用 `POST` + 动作后缀。
- 路径占位符 `:task_guid` 表示任务 GUID，`:tasklist_guid` 表示清单 GUID，需替换为实际值。
