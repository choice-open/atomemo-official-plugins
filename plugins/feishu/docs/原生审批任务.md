业务范围：原生审批任务（审批实例创建、审批实例查询与审批任务动作处理）。

## API 列表

### 审批任务管理

| 序号 | API 名称             | Method | Path                                                  | 备注 |
| ---- | -------------------- | ------ | ----------------------------------------------------- | ---- |
| 1    | 创建审批实例         | `POST` | `/open-apis/approval/v4/instances`                    |      |
| 2    | 获取单个审批实例详情 | `GET`  | `/open-apis/approval/v4/instances/:instance_id`       |      |
| 3    | 批量获取审批实例 ID  | `GET`  | `/open-apis/approval/v4/instances`                    |      |
| 4    | 同意审批任务         | `POST` | `/open-apis/approval/v4/tasks/approve`                |      |
| 5    | 拒绝审批任务         | `POST` | `/open-apis/approval/v4/tasks/reject`                 |      |
| 6    | 转交审批任务         | `POST` | `/open-apis/approval/v4/tasks/transfer`               |      |
| 7    | 退回审批任务         | `POST` | `/open-apis/approval/v4/instances/specified_rollback` |      |

## 说明

- 上述接口均来自飞书开放平台服务端 API（Approval v4）范畴，按「原生审批任务」业务场景整理。
- 原生审批实例接口位于 `approval/v4/instances` 模块。
- 审批任务动作接口位于 `approval/v4/tasks/*` 模块。
