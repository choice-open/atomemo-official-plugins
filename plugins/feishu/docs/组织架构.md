# 飞书开放平台服务端 API（组织架构）

参考文档：

- [服务端 API 列表](https://open.feishu.cn/document/server-docs/api-call-guide/server-api-list)
- [员工管理概述](https://open.feishu.cn/document/directory-v1/employee/overview)
- [部门管理概述](https://open.feishu.cn/document/directory-v1/department/overview)

业务范围：组织架构（员工管理 Employee + 部门管理 Department）。

## API 列表

### 员工管理

| 序号 | API 名称         | Method   | Path                                             | 备注                     |
| ---- | ---------------- | -------- | ------------------------------------------------ | ------------------------ |
| 1    | 创建员工         | `POST`   | `/open-apis/directory/v1/employees`              |                          |
| 2    | 更新员工信息     | `PATCH`  | `/open-apis/directory/v1/employees/:employee_id` |                          |
| 3    | 离职员工         | `DELETE` | `/open-apis/directory/v1/employees/:employee_id` |                          |
| 4    | 批量获取员工信息 | `POST`   | `/open-apis/directory/v1/employees/mget`         | 批量根据 ID 查询员工详情 |
| 5    | 批量查询员工列表 | `POST`   | `/open-apis/directory/v1/employees/filter`       | 依据条件批量获取员工列表 |
| 6    | 搜索员工信息     | `POST`   | `/open-apis/directory/v1/employees/search`       |                          |

### 部门管理

| 序号 | API 名称         | Method   | Path                                                 | 备注                     |
| ---- | ---------------- | -------- | ---------------------------------------------------- | ------------------------ |
| 7    | 创建部门         | `POST`   | `/open-apis/directory/v1/departments`                |                          |
| 8    | 更新部门         | `PATCH`  | `/open-apis/directory/v1/departments/:department_id` |                          |
| 9    | 删除部门         | `DELETE` | `/open-apis/directory/v1/departments/:department_id` |                          |
| 10   | 批量获取部门信息 | `POST`   | `/open-apis/directory/v1/departments/mget`           | 批量根据 ID 查询部门详情 |
| 11   | 获取部门列表     | `POST`   | `/open-apis/directory/v1/departments/filter`         | 依据条件批量获取部门列表 |
| 12   | 搜索部门         | `POST`   | `/open-apis/directory/v1/departments/search`         |                          |

## 说明

- 上述接口均来自飞书开放平台服务端 API 范畴，按"组织架构"业务场景整理。
- 员工管理接口位于 `directory/v1/employees` 模块。
- 部门管理接口位于 `directory/v1/departments` 模块。
