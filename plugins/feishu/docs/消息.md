## API 列表

### 消息管理

| 序号 | API 名称     | Method | Path                                 | 备注                             |
| ---- | ------------ | ------ | ------------------------------------ | -------------------------------- |
| 1    | 发送消息     | `POST` | `/open-apis/im/v1/messages`          | 向指定用户或群聊发送消息         |
| 2    | 批量发送消息 | `POST` | `/open-apis/message/v4/batch_send/`  | 向多个用户或多个部门成员发送消息 |
| 3    | 上传图片     | `POST` | `/open-apis/im/v1/images`            | 上传图片资源                     |
| 4    | 下载图片     | `GET`  | `/open-apis/im/v1/images/:image_key` | 通过 `image_key` 下载图片        |
| 5    | 上传文件     | `POST` | `/open-apis/im/v1/files`             | 上传文件资源                     |
| 6    | 下载文件     | `GET`  | `/open-apis/im/v1/files/:file_key`   | 通过 `file_key` 下载文件         |

## 说明

- 上述接口均来自飞书开放平台服务端 API 范畴，按“消息”业务场景整理。
- 消息发送接口位于 `im/v1/messages` 与 `message/v4/batch_send` 模块。
- 资源管理接口位于 `im/v1/images` 与 `im/v1/files` 模块。
