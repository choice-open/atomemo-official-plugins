业务范围：日程（Calendar Event，隶属于某一日历下的日程资源）。

## API 列表

### 日程

| 序号 | API 名称     | Method   | Path                                                               | 备注                         |
| ---- | ------------ | -------- | ------------------------------------------------------------------ | ---------------------------- |
| 1    | 创建日程     | `POST`   | `/open-apis/calendar/v4/calendars/:calendar_id/events`             | 在指定日历下创建日程         |
| 2    | 删除日程     | `DELETE` | `/open-apis/calendar/v4/calendars/:calendar_id/events/:event_id` | 删除指定日程                 |
| 3    | 更新日程     | `PATCH`  | `/open-apis/calendar/v4/calendars/:calendar_id/events/:event_id`   | 更新指定日程                 |
| 4    | 获取日程     | `GET`    | `/open-apis/calendar/v4/calendars/:calendar_id/events/:event_id`   | 获取指定日程详情             |
| 5    | 获取日程列表 | `GET`    | `/open-apis/calendar/v4/calendars/:calendar_id/events`           | 分页或增量获取某日历下的日程 |
| 6    | 搜索日程     | `POST`   | `/open-apis/calendar/v4/calendars/:calendar_id/events/search`    | 在指定日历下按条件搜索日程   |

## 说明

- 上述接口均来自飞书开放平台服务端 API 范畴，按「日程」业务场景整理。
- 日程接口均位于 `calendar/v4/calendars/:calendar_id/events` 及其子路径下。
- 路径占位符 `:calendar_id` 表示日历 ID，`:event_id` 表示日程 ID，需替换为实际值。
