# 优先实现的功能

本文列出当前迭代优先对接的飞书服务端能力，按业务域组织。实现时请以 [飞书开放平台](https://open.feishu.cn/document/server-docs/api-call-guide/server-api-list) 最新文档为准，并配合应用权限申请。

| 业务域 | 对应 SDK 模块（常见） |
| --- | --- |
| 组织架构 | `contact`（通讯录与用户、部门） |
| 消息 | `im`（即时消息、资源上传下载） |
| 日历 / 日程 | `calendar`（日历与日程事件） |
