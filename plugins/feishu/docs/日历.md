业务范围：日历（Calendar，不含日程、忙闲等子资源）。

## API 列表

### 日历

| 序号 | API 名称           | Method   | Path                                              | 备注                                       |
| ---- | ------------------ | -------- | ------------------------------------------------- | ------------------------------------------ |
| 1    | 创建共享日历       | `POST`   | `/open-apis/calendar/v4/calendars`                | 创建日历实体（含共享日历）                 |
| 2    | 删除共享日历       | `DELETE` | `/open-apis/calendar/v4/calendars/:calendar_id` | 删除指定日历                               |
| 3    | 查询主日历信息     | `POST`   | `/open-apis/calendar/v4/calendars/primary`        | 获取当前身份的主日历信息                   |
| 4    | 批量获取主日历信息 | `POST`   | `/open-apis/calendar/v4/calendars/primarys`       | 批量获取主日历；路径中 `primarys` 为原文 |
| 5    | 查询日历信息       | `GET`    | `/open-apis/calendar/v4/calendars/:calendar_id`   | 获取指定日历详情                           |
| 6    | 批量查询日历信息   | `POST`   | `/open-apis/calendar/v4/calendars/mget`           | 根据日历 ID 列表批量获取日历               |
| 7    | 查询日历列表       | `GET`    | `/open-apis/calendar/v4/calendars`                | 获取当前身份可见的日历列表                 |
| 8    | 更新日历信息       | `PATCH`  | `/open-apis/calendar/v4/calendars/:calendar_id`   | 更新指定日历属性                           |
| 9    | 搜索日历           | `POST`   | `/open-apis/calendar/v4/calendars/search`         | 按条件搜索日历                             |

## 说明

- 上述接口均来自飞书开放平台服务端 API 范畴，按「日历」业务场景整理。
- 日历相关接口位于 `calendar/v4/calendars` 与 `calendar/v4/calendars/*` 路径下；主日历相关接口见 `calendars/primary`、`calendars/primarys`。
- 路径占位符 `:calendar_id` 表示日历 ID，需替换为实际值。
