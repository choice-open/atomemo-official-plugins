# 飞书开放平台服务端 API 模块列表

飞书开放平台提供了 **2500+** 标准化服务端 API 接口，覆盖企业协同办公的各个场景。

## 模块总览

| 模块名称 | 英文名称 | API路径 | 功能描述 | 典型接口示例 |
|----------|----------|---------|----------|-------------|
| **消息与群组** | IM (Instant Messaging) | `/open-apis/im/v1` | 发送消息、群组管理、消息撤回、加急、批量消息 | 发送消息、创建群组、消息撤回、发送加急 |
| **通讯录** | Contact | `/open-apis/contact/v3` | 组织架构、用户、部门、角色、标签管理 | 获取用户、创建部门、获取角色列表 |
| **新版文档** | Docx | `/open-apis/docx/v1` | 云文档创建、编辑、块操作、权限管理 | 创建文档、获取块、插入块、文档权限 |
| **电子表格** | Sheets | `/open-apis/sheets/v3` | 电子表格创建、工作表操作、单元格管理 | 创建表格、写入单元格、公式操作 |
| **多维表格** | Bitable | `/open-apis/bitable/v1` | 数据表、字段、记录、视图、权限管理 | 列出记录、新增记录、数据表操作、权限设置 |
| **日历** | Calendar | `/open-apis/calendar/v4` | 日程管理、会议室预定、忙闲查询、请假日程 | 创建日程、查询忙闲、预定会议室 |
| **审批** | Approval | `/open-apis/approval/v4` | 审批流程、审批定义、实例管理 | 获取审批定义、发起审批、审批实例操作 |
| **知识库** | Wiki | `/open-apis/wiki/v2` | 知识库、节点、分类管理 | 创建知识库、获取节点、知识库搜索 |
| **搜索** | Search | `/open-apis/search/v1` | 用户、消息、文档、知识库搜索 | 搜索用户、搜索消息、搜索文档 |
| **服务台** | Helpdesk | `/open-apis/helpdesk/v1` | 工单、客服、标签、会话管理 | 创建工单、获取工单详情、客服会话 |
| **百科** | Baike | `/open-apis/baike/v1` | 企业百科、词条、分类、图片管理 | 搜索词条、创建词条、获取分类 |
| **飞书人事** | CoreHR | `/open-apis/corehr/v1` | 员工请假、HR数据同步 | 查询请假记录、员工信息 |
| **云空间** | Drive | `/open-apis/drive/v1` | 文件上传下载、文件夹管理、权限 | 上传文件、获取文件、文件夹操作 |
| **应用管理** | Applications | `/open-apis/applications/v1` | 应用版本、审核、可用范围管理 | 获取应用信息、获取版本列表 |
| **认证信息** | Verification | `/open-apis/verification/v1` | 企业认证状态查询 | 获取企业认证信息 |
| **个人设置** | Personal Settings | `/open-apis/personal_settings/v1` | 系统状态等个人设置 | 创建系统状态、修改系统状态 |
| **帐号** | Passport | `/open-apis/passport/v1` | 用户登录信息查询 | 批量获取脱敏用户登录信息 |
| **消息卡片** | Message Card | `/open-apis/message/v4` | 卡片消息模板、交互 | 发送卡片消息、卡片交互回调 |

## 详细功能说明

### 1. 消息与群组 (IM)

| 功能 | 说明 |
|------|------|
| 消息发送 | 支持文本、富文本、图片、文件、视频、音频、表情、卡片等消息类型 |
| 消息接收 | 订阅消息事件，接收用户发送的消息 |
| 群组管理 | 创建群、拉人入群、踢人、解散群、群信息获取 |
| 消息操作 | 消息撤回、消息编辑、消息删除、消息已读回执 |
| 消息加急 | 发送应用内加急、短信加急、电话加急 |
| 批量消息 | 批量发送消息、批量撤回、进度查询 |

### 2. 通讯录 (Contact)

| 功能 | 说明 |
|------|------|
| 用户管理 | 获取用户、创建用户、修改用户、用户批量操作 |
| 部门管理 | 获取部门、创建部门、修改部门、部门层级 |
| 角色管理 | 获取角色列表、角色成员管理 |
| 标签管理 | 用户标签创建、修改、删除 |
| 授权范围 | 获取通讯录授权范围 |

### 3. 云文档 (Docx)

| 功能 | 说明 |
|------|------|
| 文档操作 | 创建文档、获取文档、删除文档、复制文档 |
| 块操作 | 获取块、插入块、删除块、更新块 |
| 内容类型 | 文本、标题、列表、代码块、引用、分割线、图片、表格等 |
| 权限管理 | 文档协作者管理、权限转让 |

### 4. 电子表格 (Sheets)

| 功能 | 说明 |
|------|------|
| 表格操作 | 创建电子表格、获取表格信息、删除表格 |
| 工作表操作 | 创建、删除、重命名、移动工作表 |
| 单元格操作 | 读取、写入、批量写入、公式操作 |
| 筛选操作 | 创建筛选、更新筛选、获取筛选 |

### 5. 多维表格 (Bitable)

| 功能 | 说明 |
|------|------|
| App管理 | 创建多维表格、复制、删除、获取元数据 |
| 数据表操作 | 列出数据表、新增、删除数据表 |
| 字段操作 | 列出字段、新增字段、删除字段、更新字段 |
| 记录操作 | 列出记录、新增记录、批量新增、更新记录、删除记录、筛选、检索 |
| 视图操作 | 列出视图、新增视图、删除视图 |
| 权限管理 | 增加权限、移除权限、更新权限、高级权限设置 |

### 6. 日历 (Calendar)

| 功能 | 说明 |
|------|------|
| 日历管理 | 获取主日历、创建日历、获取日历列表、更新日历、搜索日历 |
| 日程管理 | 创建日程、获取日程、更新日程、删除日程、搜索日程 |
| 会议室 | 会议室列表、会议室预定 |
| 忙闲查询 | 查询用户忙闲信息 |
| 请假日程 | 创建请假日程、同步休假状态 |
| CalDAV | 生成CalDAV配置、同步日历到本地设备 |
| Exchange | 绑定/解绑Exchange账户 |

### 7. 审批 (Approval)

| 功能 | 说明 |
|------|------|
| 审批定义 | 获取审批定义、创建审批定义、关联外部选项 |
| 审批实例 | 发起审批、审批通过、审批拒绝、审批撤回 |
| 审批查询 | 查询审批实例、审批记录 |

### 8. 知识库 (Wiki)

| 功能 | 说明 |
|------|------|
| 知识库 | 创建知识库、获取知识库、删除知识库 |

### 11. 百科 (Baike)

| 功能 | 说明 |
|------|------|
| 词条管理 | 搜索词条、创建词条、编辑词条、删除词条 |
| 分类管理 | 获取分类、创建分类 |
| 图片管理 | 上传图片、下载图片 |

## API 调用前置要求

1. **创建应用**: 在飞书开发者后台创建企业自建应用或商店应用
2. **获取凭证**: 获取 `tenant_access_token` 或 `user_access_token`
3. **申请权限**: 在开发者后台申请所需 API 权限
4. **发布应用**: 创建版本并发布，通过企业管理员审核
