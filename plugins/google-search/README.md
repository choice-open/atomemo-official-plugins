# Atomemo Plugin - Google Search

在 Atomemo 工作流中调用 [Google Custom Search JSON API](https://developers.google.com/custom-search/v1/overview) 进行网页搜索。

## 使用文档

详细安装、配置与参数说明见 [docs/usage.md](./docs/usage.md)。

## Development

- Install dependencies:

```bash
bun install
```

- Run the unit tests:

```bash
bun run test
```

- Build the library:

```bash
bun run build
```
