# Delete menu (`wechat-work-delete-menu`)

Calls `GET /cgi-bin/menu/delete` to delete a custom menu for an application.

## Parameters

- **wechat_work_credential** (required): `credential_id` for `wechat-work`.
- **agent_id** (required): Numeric AgentId of the application.

## Output

Returns success status on successful deletion.
