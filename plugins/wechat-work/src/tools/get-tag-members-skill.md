# Get tag members (`wechat-work-get-tag-members`)

Calls `GET /cgi-bin/tag/get` to get the list of members in a tag.

## Parameters

- **wechat_work_credential** (required): `credential_id` for `wechat-work`.
- **tag_id** (required): Tag ID to get members from.

## Output

Returns the list of users and departments in the tag.
