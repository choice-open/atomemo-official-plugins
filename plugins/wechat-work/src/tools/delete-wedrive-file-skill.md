# Delete WeDrive file (`wechat-work-delete-wedrive-file`)

Calls `POST /cgi-bin/wedrive/file_del` to delete a file from WeDrive.

## Parameters

- **wechat_work_credential** (required): `credential_id` for `wechat-work`.
- **file_id** (required): The ID of the file to delete.

## Output

Returns success status.
