# Get meeting statistic (`wechat-work-get-meeting-statistic`)

Calls `POST /cgi-bin/meeting/statistic/get` to get meeting statistics.

## Parameters

- **wechat_work_credential** (required): `credential_id` for `wechat-work`.
- **meeting_id** (required): Meeting ID to get statistics for.

## Output

Returns meeting statistics including participant details.
