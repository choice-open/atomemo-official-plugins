# Dismiss WeDrive space (`wechat-work-dismiss-space`)

Calls `POST /cgi-bin/wedrive/space_dismiss` to dismiss (delete) a WeDrive space.

## Parameters

- **wechat_work_credential** (required): `credential_id` for `wechat-work`.
- **space_id** (required): The ID of the WeDrive space to dismiss.

## Output

Returns success status.
