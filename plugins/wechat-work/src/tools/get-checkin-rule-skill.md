# Get checkin rule (`wechat-work-get-checkin-rule`)

Calls `GET /cgi-bin/checkin/get_checkin_rule` to retrieve checkin rule for a date.

## Parameters

- **wechat_work_credential** (required): `credential_id` for `wechat-work`
- **date** (required): Date in YYYY-MM-DD format

## Output

Returns checkin rule data on success.
