# Delete kf account (`wechat-work-delete-kf-account`)

Calls `POST /cgi-bin/kf/account/del` to delete a customer service account.

## Parameters

- **wechat_work_credential** (required): `credential_id` for `wechat-work`.
- **account_id** (required): Customer service account ID to delete.

## Output

Returns success status.
