# Get JSSDK media (`wechat-work-get-jssdk-media`)

Calls `GET /cgi-bin/media/get/jssdk` to get high-definition voice/media.

## Parameters

- **wechat_work_credential** (required): `credential_id` for `wechat-work`.
- **media_id** (required): Media ID to download.

## Output

Returns the media content as base64.
