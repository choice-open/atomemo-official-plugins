# Get contact way (`wechat-work-get-contact-way`)

Calls `GET /cgi-bin/externalcontact/get_contact_way` to get the contact way configuration.

## Parameters

- **wechat_work_credential** (required): `credential_id` for `wechat-work`.
- **config_id** (required): Contact way config ID.

## Output

Returns the contact way configuration details.
