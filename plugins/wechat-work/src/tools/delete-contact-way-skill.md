# Delete contact way (`wechat-work-delete-contact-way`)

Calls `POST /cgi-bin/externalcontact/del_contact_way` to delete a contact way.

## Parameters

- **wechat_work_credential** (required): `credential_id` for `wechat-work`.
- **config_id** (required): Contact way config ID to delete.

## Output

Returns success status.
