# Get follow user list (`wechat-work-get-follow-user-list`)

Calls `GET /cgi-bin/externalcontact/get_follow_user_list` to get the list of users with customer contact enabled.

## Parameters

- **wechat_work_credential** (required): `credential_id` for `wechat-work`.

## Output

Returns the list of userids with customer contact enabled.
