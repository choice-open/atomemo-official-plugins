# Get PSTN meeting (`wechat-work-get-pstn-meeting`)

Calls `GET /cgi-bin/meeting/pstn/get` to get PSTN (phone) meeting details.

## Parameters

- **wechat_work_credential** (required): `credential_id` for `wechat-work`.
- **meeting_id** (required): Meeting ID to get PSTN details.

## Output

Returns PSTN meeting details including participants.
