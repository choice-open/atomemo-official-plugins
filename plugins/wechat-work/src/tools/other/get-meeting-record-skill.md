# Get meeting record (`wechat-work-get-meeting-record`)

Calls `GET /cgi-bin/meeting/record/get` to get meeting recording file information.

## Parameters

- **wechat_work_credential** (required): `credential_id` for `wechat-work`.
- **meeting_id** (required): Meeting ID to get recording.

## Output

Returns meeting recording file information including download URL.
