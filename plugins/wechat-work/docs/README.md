# wechat-work docs

本目录用于维护 `plugins/wechat-work` 对企业微信「企业内部开发 -> 服务端 API」的覆盖清单。

## 文档结构

- `wechat-work-server-api-modules.md`：主清单，按官方目录顺序列出 API，并在同一表中标注 `✅/⬜` 状态（不区分完成/未完成文档）。

## 维护规则

- 目录结构必须与企业微信官方文档目录保持一致。
- 新增/调整 API 时，直接在对应模块表格中更新，不另建"已完成"或"未完成"分组。
- 当插件新增工具时，同步将对应 API 的"完成情况"更新为 `✅`。
- API 描述中的 msgtype 字段表示该接口使用的消息类型。
